Ohruri
======

Japanese TrueType font for Japanese "Kaomoji".

## Component

- VL P Gothic (20140801)
- Open Sans
- Android Emoji
- DejaVu Sans
- Noto Sans Coptic
- Noto Sans Ethiopic
- Noto Sans Gurmukhi
- Noto Sans Hebrew
- Noto Sans Malayalam
- Noto Sans Tai Viet
- Noto Sans Tamil
- Noto Sans Yi
